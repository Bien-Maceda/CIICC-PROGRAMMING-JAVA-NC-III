### LearnOS
**Self-hosted PHP/MySQL Learning Management System**

A locally-hosted LMS built from the ground up and served via XAMPP. The standout feature is a *true* adaptive learning path engine — combining:
- **Bayesian Knowledge Tracing** to model per-student mastery over time
- **Collaborative filtering** to surface content based on similar learners' paths
- **Linear regression** for performance trend prediction
- **Gemini API** layered on top purely for synthesis and explanation — not as the decision-maker

Also includes custom authentication, session management, and course/content administration.
